== Description ==

For adding jquery power tooltips to your content.Use [wmpowertip][/wmpowertip]. The attributes available are 'text'(for the text to be displayed),'placement' (position of the tip options: n, e, s, w, nw, ne, sw, or se), 'followmouse'(tooltip will follow the users mouse cursor options: true/false) ,'smartplacement' (will try to keep the tip on screen options:true/false),'fadeintime'(Tooltip fade-in time in milliseconds.default : 200) ,'fadeouttime'(Tooltip fade-out time in milliseconds.default : 200) ,	'closedelay' (in milliseconds. default:100). Please see the screenshots for full details

== Installation ==

1. Upload `wm-powertip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode [wmpowertip] the content that needs power tooltip[/wmpowertip]
